
# Student CRUD API (Flask + MySQL)

## Features
- Create, Read, Update, Delete students
- Input validation
- MySQL integration

## Setup
1. Install dependencies:
   pip install -r requirements.txt
2. Configure database in db_config.py
3. Run:
   python app.py

## Endpoints
POST /students
GET /students
PUT /students/<id>
DELETE /students/<id>
